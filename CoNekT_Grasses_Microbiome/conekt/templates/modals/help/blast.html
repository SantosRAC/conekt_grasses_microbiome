BLAST

BLAST, short for Basic Local Alignment Search Tool, developed by Altschul et al. PubMed is a tool to quickly retrieve sequences similar to a query from a database. CoNekT users have two flavors of BLAST at their disposal to find sequences similar to their input.

  • blastp should be used to find sequences using a protein sequence as input.
  • blastn should be used to find sequences using a nucleotide sequence as input.
