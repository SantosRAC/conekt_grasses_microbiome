Multiple testing correction using the False Discovery Rate (FDR)

When performing a sequence of statistical tests (e.g. when testing if a group of genes is enriched for a specific GO term a test is performed for each term associated with at least one of the genes in the group) a correction needs to be applied to avoid false positives (type I errors). False discovery rate (FDR) is designed to correct for this by adjusting p-values based on the number of tests performed.
